## Directory Structure

### 1. **data_structures/** - Complete Data Structures Implementation
Organized into 4 main categories:

#### **linear/** - Linear Data Structures
- `arrays/` - Static, dynamic, multi-dimensional arrays with operations
- `linked_lists/` - Singly, doubly, and circular linked lists with traversal ops
- `stacks/` - Stack implementation and monotonic stack patterns
- `queues/` - Simple, circular, priority queues and deques
- `deques/` - Double-ended queue implementations

#### **trees/** - Tree Data Structures (13 types)
- `binary_trees/` - Basic binary tree with all traversals
- `binary_search_trees/` - BST with validation and operations  
- `avl_trees/` - Self-balancing AVL trees with rotations
- `red_black_trees/` - Red-black tree implementation
- `treaps/` - Treap (tree + heap) hybrid structures
- `splay_trees/` - Splay tree with dynamic rebalancing
- `segment_trees/` - Range query data structure with lazy propagation
- `fenwick_trees/` - Binary indexed tree for range updates/queries
- `tries/` - Prefix trees for string algorithms
- `b_trees/` - B-tree for external memory
- `sparse_tables/` - Sparse table for RMQ queries
- `cartesian_trees/` - Cartesian tree for range min queries
- `kd_trees/` - K-dimensional tree for spatial queries
- `interval_trees/`, `heavy_light_decomposition/`, `binary_lifting/`, `euler_tour_trees/`

#### **graphs/** - Graph Representations & Structures
- `representations/` - Adjacency list, matrix, edge list
- `directed_graphs/`, `undirected_graphs/`, `weighted_graphs/`
- `multigraphs/`
- `disjoint_set_union/` - Union-Find data structure

#### **hashing/** - Hashing & Hash-based Structures
- `hash_tables/` - Chaining, linear/quadratic probing, double hashing
- `rolling_hash/`, `polynomial_hash/`
- `bloom_filters/`, `skip_lists/`
- `lru_cache/`, `lfu_cache/`
- `consistent_hashing/`

### 2. **algorithms/** - Algorithm Implementations (6 paradigms)

#### **sorting/** - All Sorting Algorithms (14+ sorts)
- Comparison-based: bubble, selection, insertion, merge, quick, heap, shell, tree sort
- Non-comparison: counting, radix, bucket, pigeonhole sort
- Advanced: intro sort, tim sort, external merge sort

#### **searching/** - Search Algorithms (7 types)
- Linear search, binary search variants (first/last occurrence, rotated arrays)
- Ternary, exponential, fibonacci, jump, interpolation search

#### **recursion_backtracking/** - Recursive Algorithms
- N-queens, sudoku solver, permutations, combinations, word search, rat in maze

#### **divide_and_conquer/** - Divide & Conquer Algorithms
- Merge sort, quick sort, binary search, Strassen multiplication, closest pair

#### **greedy/** - Greedy Algorithms
- Activity selection, Huffman coding, fractional knapsack, job sequencing

#### **dynamic_programming/** - DP (11 categories, 30+ problems)
- Classic: Fibonacci, coin change, LCS, LIS, edit distance, word break
- Knapsack: 0/1, unbounded, bounded, multi-dimensional
- Matrix problems: matrix chain, max path sum, min cost path
- Digit DP, DP on trees/graphs
- Bitmask DP, profile DP, SOS DP, game theory DP
- Convex hull optimization

#### **graph_algorithms/** - Graph Algorithms (25+ algorithms)
- Traversals: BFS, DFS, bidirectional search
- Topological sort: Kahn & DFS-based
- Shortest paths: Dijkstra, Bellman-Ford, Floyd-Warshall, A*, SPFA, 0-1 BFS
- MST: Kruskal, Prim, Boruvka
- SCC: Kosaraju, Tarjan
- Biconnected components: bridges, articulation points
- Maximum flow: Ford-Fulkerson, Edmonds-Karp, Dinic, push-relabel, min-cost max flow
- Matching: bipartite, maximum, weighted, Hungarian algorithm
- Cycle detection

#### **string_algorithms/** - String Processing (15+ algorithms)
- Pattern matching: KMP, Z-algorithm, Rabin-Karp, Aho-Corasick, Boyer-Moore
- Substring: Manacher (palindrome), LRS, substring matching
- Advanced: suffix arrays, suffix trees, suffix automaton, palindromic tree
- Trie-based: trie matching, XOR queries, prefix problems

#### **bit_manipulation/** - Bit-level Operations
- Bit operations: set, unset, toggle, check
- Bitmasking: subset generation, hamming distance, bit DP
- Advanced: fast exponentiation, modular exponentiation, gray code, bit tricks

#### **number_theory/** - Number Theory (20+ topics)
- Prime numbers: sieve of Eratosthenes, segmented sieve, Miller-Rabin, factorization
- Modular arithmetic: mod add/sub/mul/exp, modular inverse
- GCD/LCM: Euclidean algorithm, extended Euclidean
- Combinatorics: permutations, combinations, factorials, Catalan numbers
- Number systems: base conversion
- Advanced: CRT, Fermat's theorem, Euler's theorem, Pollard-Rho, matrix exponentiation
